KataliaProV2 — Demo

Aplicación móvil para gestión de pólizas con roles, chat interno, métricas inteligentes y administración de clientes.

Login

Pantalla inicial de inicio de sesión con validación de correo y contraseña.

Registro

Formulario para registrar nuevos usuarios con validación de campos y confirmación de contraseña.

Gestión de Roles

Módulo para administrar roles: Administrador, Supervisor, Analista y Cliente.

Crear Tarea

Formulario para asignar tareas a usuarios, con opción de asociar vehículos y establecer fecha límite.

Gestión de Tareas

Visualización de tareas por estado (pendientes, en proceso, completadas o canceladas).

Casos de Clientes

Listado de clientes con detalle de tareas asignadas y estado de cada caso.

Chat Interno

Sistema de mensajería entre clientes, analistas, supervisores y administradores.

Recursos

Módulo para subir y descargar archivos asociados a tareas, clientes o roles.

Dashboard de Métricas

Panel de desempeño con análisis de eficiencia, tiempos promedio y clientes atendidos.
